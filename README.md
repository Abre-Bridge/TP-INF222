# Custom Blog Engine API
<span style="color:red">TP INF222</span>
A streamlined REST API for managing blog entries, built with Node.js and SQLite.

## Features

- Full CRUD operations for blog entries
- Search functionality by headline or body content
- Integrated Swagger documentation
- Built using modern ES Modules

## Project Structure

- `src/api-endpoints/`: Route definitions
- `src/handlers/`: Request processing logic
- `src/data-access/`: Database interaction layer
- `src/infrastructure/`: Core setup and documentation config
- `src/interceptors/`: Middlewares and validation

## Getting Started

1. Run `npm install`
2. Start development mode with `npm run dev` or production with `npm start`
3. Access documentation at `http://localhost:3000/docs`
